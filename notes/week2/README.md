## Quick Reference
- **Firewalls** → First line of defense (stateless vs stateful, WAF for app-layer).  
- **VPN** → Full-tunnel (all traffic via corp) vs Split-tunnel (corp only, internet direct).  
- **IDS vs IPS** → IDS detects/alerts, IPS blocks inline.
